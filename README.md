# Simple Weather App

## About
Simple weather application built with Vue 3 and OpenWeather API.

## Tech Stack
- Vue 3 (Composition API)
- OpenWeather API
- Vite
- CSS

## Installation

npm install
npm run dev

## What I learned
- Vue 3 Composition API
- State management with refs
- Working with external APIs
- Handling loading and error states
- Component communication using props and emits
